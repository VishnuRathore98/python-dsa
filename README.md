# Python DSA

A structured personnel repository for practicing and tracking Data Structures & Algorithms.

## Structure

```
python-dsa/
├── topics/ # Topic-wise problems and notes
│ ├── arrays/
│ ├── linked_list/
│ ├── trees/
│ └── ...
├── contests/ # Contest solutions
├── templates/ # Code templates
├── resources/ # Roadmaps, links, must-do lists
├── progress.md # Daily log and tracker
└── README.md
```

## Goals
- Practice daily (2+ problems/day)
- Cover all major DSA topics
- Track progress in `progress.md`

## Platforms
- [LeetCode](https://leetcode.com/)

## Notes
- Code is categorized by topic and difficulty.
- Each topic has `notes.md` and `easy/`, `medium/`, `hard/` folders.
- `resources/` contains useful lists and links.
